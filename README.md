

# IPL Data Analysis Dashboard (2008–2025)

## Overview
This project presents an interactive IPL Data Analysis Dashboard built in Microsoft Excel to analyze IPL trends, team performance, toss decisions, stadium impact, and player achievements from 2008 to 2025.

The dashboard provides dynamic filtering and visual insights using Pivot Tables, Pivot Charts, and Slicers.

---

## Dashboard Features

- Season-wise interactive filtering
- Winner and Runner-up analysis
- Player of the Season tracking
- Toss decision analysis
- Stadium-wise match trends
- Team win distribution
- Top Man of the Match performers
- Dynamic KPI visualization

---

## Key KPI & Trend Insights

### Toss Decision Analysis
- Around **61%** of teams preferred batting first after winning the toss.
- Only **39%** opted to field first.
- Stadiums like **Kingsmead** and **SuperSport Park** showed stronger batting-first success trends.

### Team Performance
- **Chennai Super Kings** and **Mumbai Indians** recorded the highest overall wins across IPL seasons.
- Team dominance remained concentrated among a few franchises over multiple years.
- **Gujarat Titans** demonstrated strong win efficiency despite fewer tournament appearances.

### Stadium Trends
- Toss strategy effectiveness varied significantly across stadiums.
- Certain venues favored batting-first outcomes while others supported chasing strategies.

### Player Performance
- Players like **KP Pietersen** and **MK Pandey** secured multiple Man of the Match awards.
- High-performing teams consistently featured strong individual performers.

### Tournament Trends
- **Royal Challengers Bangalore** appeared frequently among runner-up teams.
- Long-term consistency played a major role in franchise success.
